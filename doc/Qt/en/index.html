簡介

CIOS系統當中，經常性使用一個64位元的數值作為任意物件編碼，稱為「獨有辨識碼」（SUID, Solitary Unique Identifier）。

SUID對應到Qt的數值類型當中為quint64，對應到C/C++數值類型為unsigned long long。

Qt UUID模組負責提供一些SUID的基本操作函式及類別。

類型定義

SUID - Solitary Unique Identifier 獨有辨識碼

typedef unsigned long long SUID ; // Solitary Unique Identifier

SUID的資料大小為8位元組。

UUIDs - SUID列表

typedef QList<SUID> UUIDs ; // List of SUID
