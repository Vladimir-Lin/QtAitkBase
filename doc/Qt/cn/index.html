简介

CIOS系统当中，经常性使用一个64位元的数值作为任意物件编码，称为「独有辨识码」（SUID, Solitary Unique Identifier）。

SUID对应到Qt的数值类型当中为quint64，对应到C/C++数值类型为unsigned long long。

Qt UUID模组负责提供一些SUID的基本操作函式及类别。

类型定义

SUID - Solitary Unique Identifier 独有辨识码

typedef unsigned long long SUID ; // Solitary Unique Identifier

SUID的资料大小为8位元组。

UUIDs - SUID列表

typedef QList<SUID> UUIDs ; // List of SUID
